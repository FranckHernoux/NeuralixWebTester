Guide d'installation

Neuralix Web Tester — Application d'analyse du sommeil
https://franckhernoux.github.io/NeuralixWebTester/
Aucune installation classique n'est nécessaire. L'application fonctionne directement dans le navigateur. Vous pouvez aussi l'installer pour un accès rapide depuis le Bureau (optionnel).

1 — Utilisation rapide (sans installation)

1
Ouvrir Google Chrome, Microsoft Edge ou Brave
2
Taper l'adresse suivante dans la barre d'adresse et appuyer sur Entrée :
https://franckhernoux.github.io/NeuralixWebTester/
3
L'application est prête à l'emploi. Vous pouvez l'utiliser immédiatement.

2 — Installer sur ordinateur (recommandé)

1
Ouvrir Google Chrome, Edge ou Brave et aller à l'adresse ci-dessus
2
Attendre que la page se charge complètement (quelques secondes)
3
Cliquer sur l'icône d'installation dans la barre d'adresse (petit écran avec une flèche ↓), puis cliquer sur « Installer »
Si l'icône n'apparaît pas : cliquer sur les 3 points du menu (⋮) → « Installer Neuralix Web Tester »
4
C'est terminé ! Un raccourci apparaît sur le Bureau et dans le menu Démarrer. L'application s'ouvre dans sa propre fenêtre.
L'application se met à jour automatiquement à chaque ouverture, sans aucune action de votre part.

3 — Installer sur téléphone ou tablette

Android (Chrome)

1
Ouvrir Chrome et aller à l'adresse ci-dessus
2
Appuyer sur les 3 points (⋮) en haut à droite
3
Appuyer sur « Ajouter à l'écran d'accueil »
4
Confirmer → l'icône apparaît comme une application normale

iPhone / iPad (Safari)

1
Ouvrir Safari et aller à l'adresse ci-dessus
2
Appuyer sur l'icône de partage (carré avec flèche ↑)
3
Faire défiler et appuyer sur « Sur l'écran d'accueil »
4
Confirmer → l'icône apparaît sur l'écran d'accueil
Note : Sur iPhone/iPad, seul Safari permet l'installation. Chrome et Firefox sur iOS ne proposent pas cette option.

4 — Désinstaller

×
Ordinateur : Clic droit sur l'icône → Désinstaller
ou dans l'application : 3 points (⋮) → Désinstaller
×
Téléphone : Appui long sur l'icône → Supprimer
Navigateurs compatibles : Google Chrome, Microsoft Edge, Brave, Samsung Internet.
Firefox ne supporte pas l'installation sur ordinateur (mais l'application fonctionne dans le navigateur).
