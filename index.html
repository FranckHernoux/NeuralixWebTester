Analyse des erreurs consécutives

Qu'est-ce que cette analyse ?

Détecte les séquences d'erreurs consécutives (bursts) : plusieurs epochs mal classées d'affilée.

Les indicateurs

  • Erreurs totales : nombre d'epochs incorrectes.
  • Séquences d'erreurs : groupes consécutifs.
  • Erreurs isolées : erreurs ponctuelles (1 epoch).
  • Séquences ≥ 5 : longues séries, souvent liées aux transitions mal gérées.
  • Plus longue séq. : pire cas de confusion prolongée.

L'histogramme

Barres bleues = courtes (< 5), rouges = longues (≥ 5). Un bon modèle a surtout des erreurs isolées.

Pourquoi c'est utile ?

Erreurs isolées = bruit normal. Longues séquences = zones où le modèle décroche (confusion N1/N2, transitions).
